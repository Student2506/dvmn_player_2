# Видео плеер, стилизованный по ТЗ
Данный проект является обучающим проектом по верстке "стороннего" видео плеера

## Как установить
Данные компонент не требует установки. Достаточно скопировать на жёсткий диск и запустить index.html

## Цель проекта
Код написан в образовательных целях на онлайн-курсе для веб-разработчиков dvmn.org.
